#  React Notes App

A simple and elegant Notes Application built using **React.js**, as part of my **MEARN Stack training at Luminar Technolab**.  
This project focuses on clean UI, smooth functionality, and core React concepts like state management and props.

---

##  Features

✅ Add new notes  
✅ Delete notes instantly  
✅ Notes persist using **LocalStorage**  
✅ Clean, modern **blue-themed UI**  
✅ Responsive and user-friendly layout  

---

##  Tech Stack

- **Frontend:** React.js  
- **Styling:** CSS, Material UI  
- **Storage:** LocalStorage
